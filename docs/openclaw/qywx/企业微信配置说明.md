## 🔒 访问控制说明

### 私聊访问策略 (`dmPolicy`)
- **"open"** (当前设置): 所有用户都可以发送私聊消息，无需审批
- **"pairing"**: 需要配对审批
- **"allowlist"**: 只允许白名单用户
- **"disabled"**: 禁用所有私聊

### 群聊访问策略 (`groupPolicy`)
- **"open"** (当前设置): 允许所有群聊消息
- **"allowlist"**: 只允许白名单群组
- **"disabled"**: 禁用所有群聊消息

## 📝 使用说明

### 测试连接
在企业微信中直接向Bot发送消息：
```
你好
```

如果配置正确，OpenClaw应该会回复消息。

### 主动发送消息
```bash
# 向用户发送消息
openclaw send --channel wecom --to <用户ID> "你好"

# 向群组发送消息
openclaw send --channel wecom --to chat:<群组ID> "群消息测试"
```

### 查看配对请求
```bash
# 查看待处理的配对请求
openclaw pairing list wecom

# 批准配对请求
openclaw pairing approve wecom <配对码>
```
